
Q1: Why might we consider using Git?

Answer: We might consider using Git to organize all our code in a standardized way on a platform like Github, which is compatible with Git. In addition, in a team setting, Git is fantastic for code collaboration, as you can share repos and work on code at the same time with others.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science could be applied to biology. Within biology, there a large datasets of DNA strands and genetic combinations that can be analyzed through data science techniques, as going through those combinations one by one wouldn't be optimal.

Q3: What is the difference between Merging and Rebasing?

Answer: With merging, when you merge the two branches together, it keeps the original commit history of both the branches in the commit log. With rebasing, it rewrites the commit history in such a way where you can't tell that there was a "merge" between two branches.
